# GD&T

## Purpose
Record datum strategy and geometric control choices so manufacturing and inspection intent stays clear.

## Key rules
- Define functional datums first, then assign controls tied to assembly function.
- Avoid over-constraining with redundant geometric tolerances.
- Use feature control frames that inspection can execute directly.

## Worked example
A mounting face can be `Datum A`, with two hole centers as `Datum B` and `Datum C`, then profile controls applied to mating surfaces.
