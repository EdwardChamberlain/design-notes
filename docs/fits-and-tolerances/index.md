# Fits and Tolerances

## Purpose
Track preferred fit classes and tolerance allocation decisions for mating components.

## Key rules
- Pick fit type from function first: clearance, transition, or interference.
- Reserve tight tolerances for true function drivers.
- Stack tolerances early for high-risk interfaces.

## Worked example
A rotating shaft in a housing may start with a clearance fit pair, then be tightened only if runout or noise targets fail.
