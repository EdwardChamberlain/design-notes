# Metrology

## Purpose
Capture inspection strategy, gauge selection, and measurement uncertainty assumptions.

## Key rules
- Align gauge resolution with tolerance width.
- Define datum simulation clearly for repeatable setups.
- Record calibration status for all critical measurement tools.

## Worked example
A bore with a tight positional tolerance may require CMM inspection against datums `A|B|C`, while non-critical dimensions stay on calipers.
