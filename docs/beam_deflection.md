# Beam Deflection

## Beam Deflection Formula

![Beam Deflection](./assets/images/line_drawings/cantilever_beam_deflection_formulas.png_dark_mode.png)


| Beam and load case | Maximum beam deflection |
|---|---|
| Cantilever with point load at free end | $\delta_{\max} = \dfrac{PL^3}{3EI}$ |
| Cantilever with point load at distance \(a\) from the fixed end | $\delta_{\max} = \dfrac{Pa^2(3L-a)}{6EI}$ |
| Cantilever with uniformly distributed load | $\delta_{\max} = \dfrac{wL^4}{8EI}$ |
| Cantilever with triangular load, maximum at fixed end | $\delta_{\max} = \dfrac{wL^4}{30EI}$ |
| Cantilever with triangular load, maximum at free end | $\delta_{\max} = \dfrac{11wL^4}{120EI}$ |
| Cantilever with applied end moment | $\delta_{\max} = \dfrac{ML^2}{2EI}$ |
