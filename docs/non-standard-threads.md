# Threads - Non Standard
## Rules of Thumb
| Feature | Rule of thumb (CAD) |
|---|---|
| **Internal thread: tap drill (cutting tap)** | **Tap drill ≈ D − P** |
| **Internal thread: major-diameter clearance relief** | **Relief Ø ≥ D + 0.2 mm**, **depth ≈ 1–2P** |
| **External thread: major diameter (OD)** | **Model OD = d (nominal)** |
| **External thread: relief groove root diameter** | **Relief Ø ≤ d − 1.3P** (safe margin) |

## External threads (male)
### Major diameter (OD)
- Model major diameter at **nominal**: **d**
- Example: **M8×1 → Ø8.0 mm**

### External thread relief groove (witness-free runout)
Purpose: gives the threading tool somewhere to exit so no partial thread remains at a shoulder.

- Choose relief groove **root diameter ≤ external minor diameter**
- Handy rule: **relief Ø ≤ d − 1.3P** (safe margin below basic minor)
- Example: **M8×1 → d₃ ≈ 6.77 mm → relief Ø ≈ 6.7 mm**

## Internal threads (tapped holes)
### Tap drill (cutting tap)
- **Tap drill ≈ D − P**
- Example: **M8×1 → 8 − 1 = 7.0 mm**

### Internal thread relief (major-diameter clearance)
Purpose: lets a mating male thread seat without fouling on imperfect first threads / runout.

- **Counterbore/relief diameter ≥ D + 0.2 mm** (typical clearance)
- **Relief depth ≈ 1–2P** (increase if you need full seating near a shoulder)
- Example: **M8×1 → Ø ≥ 8.2 mm, depth ≈ 1–2 mm**

## Quick example (M8×1)
- Tap drill: **Ø7.0 mm** [8 - 1]
- Male OD: **Ø8.0 mm** (Nominal)
- Male relief groove root: **Ø6.7 mm** [8 - (1.3 * 1)]
- Internal major-clearance relief: **Ø8.2 mm × 1–2 mm deep** [8 + 0.2]
