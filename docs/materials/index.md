# Materials

## Purpose
Summarize material selection rationale, critical properties, and process compatibility notes.

## Key rules
- Document mechanical and thermal requirements before choosing grade.
- Include environment limits (temperature, corrosion, fatigue cycle count).
- Keep substitutions tied to verified equivalence criteria.

## Worked example
For a lightweight bracket, compare `6061-T6` and `7075-T6` against yield strength, corrosion exposure, and machinability before release.
