# Useful Drafting Notes

## Generic Drawing Notes

### Make From Model
```
MAKE FROM MODEL.
GENERAL SPECIFICATIONS IN ACCORDANCE WITH ISO 22081.
GENERAL LINEAR AND ANGULAR TOLERANCES TO ISO 2768 CLASS m.
DRAWING DEFINES CRITICAL REQUIREMENTS.
```

### General Tolerances
```
UNLESS OTHERWISE SPECIFIED, TOLERANCES ON DIMENSIONS WITHOUT INDIVIDUAL TOLERANCE VALUES ARE ACCORDING TO DIN ISO 2768-MK.
```

## Manufacturing Method
### MJF
```
ADDITIVELY MANUFACTURED PART.
MANUFACTURE FROM STL FILE.
INSPECTION DIMENSIONS SHOWN.
MATERIALS:
    HP PA12 GLASS BEADS
    FUSING AGENT
    DETAILING AGENT
```

### DMLS
```
ADDITVELY MANUFACTURED PART.
PROCESS: DIRECT METAL LASER SINTERING (DMLS).
MANUFACTURE FROM STL / CLI FILE.
INSPECTION DIMENSIONS SHOWN.
```

### Printed Supports
```
BREAKAWAY SUPPORTS PERMISSIBLE ON FACES MARKED 'X X X'. MAX PROJECTION 0.1 MM. REMOVE SHARP EDGES.
```

### Lattices
```
LATTICE DOMAIN SHOWN AS SOLID BODY. REFER TO SOLID MODEL FOR LATTICE GEOMETRY.
```

### Clean
```
PARTS TO BE SUPPLIED SUFFICIENTLY CLEANED AND FREE OF OIL OR SWARF.
```

### No Auto Debur
```
DO NOT AUTO-DEBUR.
```

### Deburr Sharp Edges
```
AUTO-DEBUR TO REMOVE ALL SHARP EDGES.
```
