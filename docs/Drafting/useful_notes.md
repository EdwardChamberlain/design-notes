# Useful Drafting Notes

## Generic Drawing Notes

### Make From Model
```
MAKE FROM MODEL. GENERAL SPECIFICATIONS IN ACCORDANCE WITH ISO 22081. GENERAL LINEAR AND ANGULAR TOLERANCES TO ISO 2768-1 CLASS m.
```

### General Tolerances
```
Unless otherwise specified, variations on dimensions without individual tolerance values are according to DIN ISO 2768-mk.
```
