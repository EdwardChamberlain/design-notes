# Alt Codes and NX Codes

## Alt Codes
### Common
<!-- table-class: table-layout-auto -->
| Symbol | Alt Code | Typical Use |
|---|---|---|
| Ø | 0216 | Diameter |
| ° | 248 | Degrees, temperature |
| ± | 241 | Tolerance |
| µ | 230 | Microns, micro prefix |
| × | 158 | Multiplication, dimensions |
| ² | 0178 | Squared units, e.g. mm² |
| ³ | 0179 | Cubed units, e.g. mm³ |
