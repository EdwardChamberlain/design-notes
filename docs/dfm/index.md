# DFM

## Purpose
Document manufacturability constraints early to reduce iteration cycles and cost surprises.

## Key rules
- Prefer geometry that matches standard tool access and process windows.
- Flag features with high scrap risk or long cycle impact.
- Include supplier feedback in every major design review.
