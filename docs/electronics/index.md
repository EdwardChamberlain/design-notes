# Electronics

## Purpose
Capture interface assumptions, power budgets, and integration notes for electro-mechanical subsystems.

## Key rules
- Track nominal and worst-case voltage/current limits.
- Reserve margin for thermal rise and transient events.
- Keep connector pinouts and grounding strategy versioned with hardware revisions.

## Worked example
A sensor rail budget can be tabulated with nominal and peak current to verify regulator headroom before PCB release.
