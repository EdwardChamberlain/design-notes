# Metric Threads

## Purpose
Capture quick design checks and notation for ISO metric thread decisions.

## Key rules
- Match pitch to load case and assembly speed.
- Keep coarse pitch for robustness unless fine pitch is needed for adjustment or thin walls.
- Verify thread engagement against parent material strength.

## Worked example
Given lead `L = 7.5 mm` over `N = 5` turns, pitch is:

$$
p = \frac{L}{N} = \frac{7.5}{5} = 1.5\ \mathrm{mm}
$$

For an `M10 x 1.5` thread, a quick reference set is:

| Attribute | Symbol | Value |
| --- | --- | --- |
| Nominal diameter | `d` | 10 mm |
| Pitch | `p` | 1.5 mm |
| Approx. minor diameter (internal, rough check) | `d_1` | ~8.5 mm |
