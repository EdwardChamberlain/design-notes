# Threads

## Purpose
Define thread selection and specification rules for fasteners and tapped features.

## Key rules
- Use ISO metric threads by default unless a supplier constraint requires otherwise.
- Specify full callout format: diameter, pitch, tolerance class, and engagement length.
- Check stripping and tensile failure modes at the concept stage.

## Worked example
A typical callout for a general-purpose tapped hole is `M8 x 1.25 - 6H` with a note for minimum thread depth.
